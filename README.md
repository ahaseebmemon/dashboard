# Dashboard Management System

A production-ready administrative dashboard built to manage organizations and scale team memberships. This project was developed to demonstrate a modern, serverless architecture using React, TypeScript, and Supabase.

## 🚀 Live Environments

We utilize a two-branch Git workflow (`main` for production, `development` for staging):
- **Production URL (`main` branch):** dashboard-it4o45uo2-hm007.vercel.app
- **Preview URL (`development` branch):** dashboard-81vpc7gt6-hm007.vercel.app

## 🛠 Required Tech Stack
- **Frontend:** React 18, TypeScript (Strict Mode), Vite
- **Routing:** React Router v6 (with Protected Routes)
- **Styling & UI:** Tailwind CSS, shadcn/ui, Lucide React
- **State Management:** TanStack React Query (Zero raw `useEffect` data fetching)
- **Forms & Validation:** React Hook Form + Zod validation schemas
- **Backend & Database:** Supabase (PostgreSQL, Auth, Edge Functions)
